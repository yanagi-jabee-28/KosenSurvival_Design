# KosenSurvival UI Layout Design

更新日: 2025-12-25

---

## 概要

テキストノベル（ADV）要素とゲームシステムの両立を目指し、以下の領域を設計します。

## 領域定義

### 1. **Header**
- **高さ**: 60px（デスクトップ）/ 50px（モバイル）
- **コンテンツ**: ロゴ/タイトル、メニューボタン（ハンバーガー）
- **背景**: ダークブルー（#1f6feb）

### 2. **Main Content Area**
- **デスクトップ**: 左側、幅 680px
- **モバイル**: 全幅
- **構成**:
  - **Background** (Optional): 教室/寮/図書館など背景画像（高さ 300px）
  - **Character Stand**: キャラクタ立ち絵（左/中央/右配置、高さ 250px）
  - **Text Panel**: ナレーション/会話テキスト（高さ 100-120px、スクロール可能）
  - **Text Controls**: 「次へ」「最初から」「自動送り」などのボタン（高さ 40px）
  - **Command Buttons**: 行動選択ボタン（3-4個、グリッド、高さ 50px x 2行）

### 3. **Quick Status Panel**（右コラム、デスクトップのみ）
- **幅**: 280px
- **固定位置**: 右側、上部に sticky
- **コンテンツ**:
  - 体力バー
  - メンタルバー
  - 資金表示
  - Logic / Charisma 簡易表示
  - 「ステータス詳細」リンク
  - 「メニュー」リンク

### 4. **Footer**
- **高さ**: 40px
- **コンテンツ**: セーブ/ロード、設定、著作権等
- **背景**: ライトグレー（#fafafa）

---

## デザイン原則

1. **テキストノベル要素を優先**: メイン領域でテキストと立ち絵が視覚的に統合
2. **ステータスは常時表示**: 右サイドパネル（デスクトップ）or トップステータスバー（モバイル）
3. **コマンド=アクション**: 行動ボタンはテキスト直下で視認性を確保
4. **メニューはサイドに**: デスクトップで右サイド、モバイルでハンバーガー
5. **レスポンシブ**: 960px ブレークポイント

---

## CSS Grid/Flex 方針

### デスクトップレイアウト（960px+）
```css
body {
  display: grid;
  grid-template-rows: 60px 1fr 40px;
  grid-template-columns: 1fr 280px;
  min-height: 100vh;
}

header { grid-column: 1 / -1; }
main { grid-column: 1; grid-row: 2; }
.sidebar { grid-column: 2; grid-row: 2; position: sticky; top: 60px; }
footer { grid-column: 1 / -1; }
```

### モバイルレイアウト（<960px）
```css
body {
  display: grid;
  grid-template-rows: 50px auto 1fr;
  grid-template-columns: 1fr;
  min-height: 100vh;
}

header { grid-row: 1; }
status-bar { grid-row: 2; }
main { grid-row: 3; }
footer { grid-row: 4; }
```

## 立ち絵配置戦略

### 立ち絵スロット（デスクトップ）
- **左（L）**: 高さ 250px、幅 200px（左寄せ）
- **中央（C）**: 高さ 280px、幅 220px（センター）
- **右（R）**: 高さ 250px、幅 200px（右寄せ）
- **背景**: グラデーション or 白フェード

### 立ち絵スロット（モバイル）
- **中央（C）のみ**: 高さ 150px、幅 100px

## ステータス詳細ビュー（モーダル）

- **トリガー**: 右サイドパネルの「ステータス詳細」リンク
- **コンテンツ**:
  - 全パラメータ（体力/メンタル/資金/地頭/人望/業）バー表示
  - 科目進度（複数科目を折り畳み/展開）
  - ルートフラグ進捗
  - クローズボタン
