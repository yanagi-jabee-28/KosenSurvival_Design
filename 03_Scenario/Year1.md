# Year 1: The Filter (淘汰の年) - Weekly Chart

## 1. 概要
*   **システム**: 週間スケジュール制 (平日オート/休日マニュアル)
*   **目標**: 32週を生き残り、進級すること。

## 2. 1Q: 洗礼 (Apr - June)
環境適応の時期。授業スピードに慣れるのが最優先。

| 週 | イベント名 | 内容・判定 |
|:---|:---|:---|
| **W1** | **入学オリエンテーション** | [Manual] 諏訪野から「監獄のルール」を聞かされる。部活見学開始。 |
| **W2** | **授業開始** | [Auto] 最初の授業。理解度(Liquid)が一気に増える。 |
| **W3** | **部活入部** | [Auto] Aルート(ミナ) / Dルート(生徒会) のフラグ立ち。 |
| **W4** | **黄金週間 (GW)** | [Manual] **大型連休**。遊ぶか予習するかで五月病デバフ判定。 |
| **W5** | **五月病** | [Auto] GWに遊んだ場合、Sanity減少量2倍。 |
| **W6** | **小テスト** | [Auto] 軽いジャブ。Liquidスコアの確認。 |
| **W7** | **テスト発表** | [Auto] **「テスト期間」突入**。部活停止。BGM変化。 |
| **W8** | **前期中間試験** | **[Boss Battle]** 初の試験。一夜漬けの威力を知る。 |

## 3. 2Q: 選択 (June - Sept)
中だるみと湿気の時期。タケシの運命が決まる。

| 週 | イベント名 | 内容・判定 |
|:---|:---|:---|
| **W9** | **試験返却** | [Auto] 赤点なら「補習」フラグ。順位発表(アイとの差を知る)。 |
| **W10** | **梅雨** | [Auto] 雨でSanityが削れる。室井(仙人)の部屋に入り浸る誘惑。 |
| **W11** | **レポート課題提示** | [Auto] 膨大な量の実験レポートが出される。 |
| **W12** | **中だるみ** | [Auto] 全員の勉強効率ダウン。タケシとゲーセンに行くイベント。 |
| **W13** | **レポート締切前** | [Manual] **修羅場**。ミナのレポートを手伝うか(Aルート分岐)。 |
| **W14** | **期末テスト発表** | [Auto] タケシが「今回マジでやばい」と言い出す。 |
| **W15** | **運命の分岐点** | [Manual] **タケシ救済イベント**。自分の勉強時間を割くか？ |
| **W16** | **前期期末試験** | **[Boss Battle]** 難易度上昇。タケシの点数も裏で計算される。 |

## 4. Summer Vacation (Aug - Sept)
*   **行動**: 4週分の行動をまとめて選択。
    *   **[帰省]**: Sanity全快。
    *   **[バイト]**: Cルート資金調達。
    *   **[補習]**: 赤点者の強制収容所。
*   **結末**: 始業式で「タケシの席」があるかどうかが判明する。
